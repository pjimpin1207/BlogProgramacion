Programas básicos con Python

Ejercicios prácticos para aprender la lógica del lenguaje Python

Inicio Tema 1 GitHub

Publicado por pjimpin1207 el 25 de septiembre de 2025

En este post encontrarás una recopilación de programas básicos escritos en Python, ideales para practicar los fundamentos del lenguaje y entender la lógica de programación paso a paso.

1. Programa que pide tu nombre y saluda

Código:

nombre = input("Introduce tu nombre: ")
print("Hola " + nombre)

Ejecución:

Introduce tu nombre: Pablo
Hola Pablo

2. Programa que muestra "Hola nombre" directamente

Código:

nombre = "Pablo"
print("Hola " + nombre + "!")

Ejecución:

Hola Pablo!

3. Suma de dos números introducidos por teclado

Código:

num1 = int(input("Introduce el primer número: "))
num2 = int(input("Introduce el segundo número: "))
suma = num1 + num2
print("La suma es:", suma)

Ejecución:

Introduce el primer número: 5
Introduce el segundo número: 8
La suma es: 13

4. Verificación de usuario

Código:

usuario_correcto = "admin"
usuario = input("Introduce tu usuario: ")

if usuario == usuario_correcto:
    print("Bienvenido,", usuario)
else:
    print("Error de inicio de sesión")

Ejecución:

Introduce tu usuario: admin
Bienvenido, admin

5. Mayor o menor (y comprobar si son iguales)

Código:

a = int(input("Introduce el primer número: "))
b = int(input("Introduce el segundo número: "))

if a > b:
    print("El mayor es:", a)
elif b > a:
    print("El mayor es:", b)
else:
    print("Son iguales")

Ejecución:

Introduce el primer número: 7
Introduce el segundo número: 7
Son iguales

6. Número par o impar

Código:

num = int(input("Introduce un número: "))

if num % 2 == 0:
    print("El número es par")
else:
    print("El número es impar")

Ejecución:

Introduce un número: 6
El número es par

7. Operaciones básicas (suma, resta, multiplicación, división)

Código:

a = int(input("Introduce el primer número: "))
b = int(input("Introduce el segundo número: "))

print("Suma:", a + b)
print("Resta:", a - b)
print("Multiplicación:", a * b)
print("División:", a / b)

Ejecución:

Introduce el primer número: 6
Introduce el segundo número: 3
Suma: 9
Resta: 3
Multiplicación: 18
División: 2.0

8. Año bisiesto

Código:

anio = int(input("Introduce un año: "))

if (anio % 4 == 0 and anio % 100 != 0) or (anio % 400 == 0):
    print("El año es bisiesto")
else:
    print("El año no es bisiesto")

Ejecución:

Introduce un año: 2024
El año es bisiesto

9. Calcular nota

Código:

nota = float(input("Introduce tu nota: "))

if nota >= 5:
    print("Aprobado")
else:
    print("Suspenso")

Ejecución:

Introduce tu nota: 6.5
Aprobado

10. Calculadora básica

Código:

a = float(input("Introduce el primer número: "))
b = float(input("Introduce el segundo número: "))

print("Operaciones disponibles: +, -, *, /")
op = input("Selecciona una operación: ")

if op == "+":
    print("Resultado:", a + b)
elif op == "-":
    print("Resultado:", a - b)
elif op == "*":
    print("Resultado:", a * b)
elif op == "/":
    print("Resultado:", a / b)
else:
    print("Operación no válida")

Ejecución:

Introduce el primer número: 8
Introduce el segundo número: 2
Selecciona una operación: *
Resultado: 16

11. Tipo de triángulo

Código:

a = int(input("Introduce el lado A: "))
b = int(input("Introduce el lado B: "))
c = int(input("Introduce el lado C: "))

if a == b == c:
    print("Triángulo equilátero")
elif a == b or b == c or a == c:
    print("Triángulo isósceles")
else:
    print("Triángulo escaleno")

Ejecución:

Introduce el lado A: 5
Introduce el lado B: 5
Introduce el lado C: 8
Triángulo isósceles

12. 100 líneas de "Hola Mundo"

Código:

for i in range(1, 101):
    print("Hola Mundo", i)

Ejecución:

Hola Mundo 1
Hola Mundo 2
...
Hola Mundo 100
